
Accepted tokens:

* `#define`
* `#if`
* `#elif`
* `#endif`
* `#else`
* `#ifdef`
* `#ifndef`
* `#include` - the same as `@require`
* `#import` - the same as `@require_once`
* `@require` 
* `@require_once`
* `@using`

**IMPORTANT:** `#undef` and `#pragma` are not supported

* `{{expression}}` - expands a Smarty expression
